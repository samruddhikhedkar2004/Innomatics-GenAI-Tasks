# 🧠 Assignment 1: NLP Preprocessing Engine

## 📌 Overview
This project focuses on building a robust NLP preprocessing pipeline to clean and transform raw text data into meaningful tokens.

The pipeline handles real-world noisy data including emojis, URLs, repeated characters, and irregular patterns.

---

## 🚀 Features
- Text normalization (lowercasing, removing noise)
- Removal of numbers, URLs, and special characters
- Handling repeated characters
- Tokenization
- Removal of short and irrelevant words
- Token analytics and frequency analysis
- Error handling for edge cases

---

## 🛠 Tech Stack
- Python
- Regular Expressions (re)
- Collections (Counter)

---

## ✅ Conclusion
In this project, I developed a complete NLP preprocessing pipeline capable of handling noisy real-world text data.

The solution includes cleaning, tokenization, analysis, and error handling, making it suitable for further NLP and machine learning applications.

Hence, this preprocessing pipeline serves as a strong foundation for building efficient NLP systems.
