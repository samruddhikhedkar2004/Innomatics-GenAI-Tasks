#  Assignment 1: NLP Preprocessing Engine

## 📌 Overview
This project focuses on building a robust NLP preprocessing pipeline to clean and transform raw text data into meaningful tokens.

The pipeline handles real-world noisy data including emojis, URLs, repeated characters, and irregular patterns.

---

## 🚀 Features
- Text normalization (lowercasing, removing noise)
- Removal of numbers, URLs, and special characters
- Handling repeated characters
- Tokenization
- Removal of short and irrelevant words
- Token analytics and frequency analysis
- Error handling for edge cases

---

## 🛠 Tech Stack
- Python
- Regular Expressions (re)
- Collections (Counter)

---

## ✅ Conclusion
In this project, I developed a complete NLP preprocessing pipeline capable of handling noisy real-world text data.

The solution includes cleaning, tokenization, analysis, and error handling, making it suitable for further NLP and machine learning applications.

Hence, this preprocessing pipeline serves as a strong foundation for building efficient NLP systems.

# Assignment 2: Sentiment Analysis using NLP Pipeline & ML Models

## 📌 Overview

This project focuses on building a complete end-to-end Sentiment Analysis system using Natural Language Processing (NLP) techniques and Machine Learning models.

The objective is to transform raw text data into meaningful features and use them to classify sentiments as positive or negative.

---

## 🎯 Objectives

* Understand text preprocessing in NLP
* Convert text data into numerical features
* Train and evaluate multiple ML models
* Compare model performance using evaluation metrics

---

## 🚀 Features

* Text preprocessing (lowercasing, punctuation removal, stopwords removal)
* Handling real-world noisy data (HTML tags, special characters, URLs)
* Tokenization and stemming
* Feature Engineering using Bag of Words (BoW) and TF-IDF
* Training multiple ML models
* Model evaluation using Accuracy, Precision, Recall, and F1 Score
* Model performance comparison using visualization

---

## 🛠 Tech Stack

* Python
* Jupyter Notebook (VS Code)
* Pandas, NumPy
* NLTK
* Scikit-learn
* Matplotlib

---

## 📊 Workflow

Raw Data → Preprocessing → Feature Engineering → Model Training → Evaluation → Comparison

---

## 📈 Models Used

* Logistic Regression
* Naive Bayes
* Decision Tree

---

## 📊 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score

---

## 🔍 Key Insights

* TF-IDF provided better feature representation compared to Bag of Words.
* Logistic Regression achieved the best performance among all models.
* Naive Bayes was faster but slightly less accurate.
* Decision Tree showed lower performance due to possible overfitting.

---

## ▶️ How to Run

1. Open the notebook in VS Code or Jupyter Notebook
2. Install required libraries
3. Run all cells step-by-step
4. View outputs and visualizations

---

## 📌 Dataset

* IMDb Movie Reviews Dataset (Kaggle)

---

## ✅ Conclusion

In this project, we successfully built a sentiment analysis pipeline using NLP and machine learning techniques.

The results show that proper preprocessing and feature engineering significantly improve model performance.

Hence, this project demonstrates an effective approach for solving real-world text classification problems.
